
# Pixel Platformer (GitHub Pages)

## Controls
- Move: Arrow keys or A/D
- Jump: Space
- Throw weapon (home phone): F
- Dash: Shift (after you pick up the Dash powerup)

## Setup
1. Put all game files in the repo root.
2. Put all PNGs in /assets with exact filenames:
   - Background_Pic.png
   - Platform.png
   - Exit_Door.png
   - CheckpointFlag.png
   - Coin.png
   - Enemy1.png
   - Enemy2.png
   - Gilly.png
   - Scott.png
   - Kevin.png
   - Nate.png
   - Powerup_Dash.png
   - Powerup_Speedboost.png
   - powerup_homephone.png
